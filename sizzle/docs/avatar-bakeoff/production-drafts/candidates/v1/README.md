# Candidate V1 Workspace

This folder is reserved for the next serious avatar asset pass.

V1 should contain only assets that are intended to be reviewed as candidates, not diagnostic mask-fill outputs from V0.

Initial V1 target:

- Greybox appearance lock: medium skin, long straight hair, blue eyes.
- Body candidates for the wider palette: light, medium, dark.
- Cleaned `hairBack` / `hairFront` masks for long straight hair.
- Separate `face`, `eyes`, `expression/brows`, `expression/mouth`, `nipples`, and `genitals` layers.
- Clean garment-only underwear extraction.
- QA composites for every accepted candidate layer family.

Current body candidates:

- `body/20_body-light-comfy-00029.png` - user-selected pale/light candidate.
- `body/20_body-medium-canonical-noface.png` - canonical medium noface body candidate.
- `body/20_body-dark-comfy-00017.png` - user-selected darker/dusky candidate.

Earlier duplicate names such as `20_body-light-pale-comfy-00029.png` and `20_body-medium-dusky-comfy-00017.png` may remain for provenance, but the three filenames above are the active candidate names.

Do not promote from here to `sizzle/media/avatar/` until crossed composites pass review.
