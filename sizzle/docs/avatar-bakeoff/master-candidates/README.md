## Review Criteria

Pick the master candidate that best supports:

- Stable front-facing body geometry.
- Arms relaxed with gaps for clothing extraction.
- Clear hair silhouette.
- Face/eye/brow/mouth regions that can support expression overlays.
- Underwear baseline suitable for extracting both body and clothing layers.
- Stylized realism rather than full photographic realism.
- Sizzle tone: mature, grounded, 2005 Toronto thriller, adult but not pornographic.
