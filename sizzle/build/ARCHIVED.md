# ARCHIVED — Tweego build scripts (retired 2026-07-11)

SugarCube was retired at Phase 6 of `sizzle/docs/GODOT-PORT-PLAN.md`; the Godot project
(`godot/`) is the build now. These scripts compiled the twee source that has moved to
`sizzle/archive/twee-src/` — they are kept for the historical record and no longer run
as-is. The twee line lives on in a fork and on the frozen `master` branch.
